# Meme Hunter AI

Paper-trading dashboard for newly listed Solana/Pump.fun-style meme coins using DexScreener data.

## Run locally / VPS

```bash
cd MemeHunterAI
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

Open:

```text
http://YOUR_SERVER_IP:5000
```

## Important

This starts in paper-trading mode only. It does not connect to a wallet or place real trades.
